## Overview

This repository contains the code and documentation for a Master's research project that combines machine learning and combinatorial optimization to address school dropout in Brazil.

The pipeline predicts individual dropout risk using census microdata (Censo da Educação Básica, 2019–2024), estimates each student's responsiveness to educational interventions, and then solves a resource allocation problem to maximize dropout reduction under budget constraints.

